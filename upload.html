GitHub API

Upload file to this repo

Use the official GitHub REST API to add files (e.g. PDFs under files/) without leaving this site.

The GitHub API itself是免费的，只要在免费额度内使用即可。下面的表单在浏览器中直接调用 GitHub 的 /contents 接口，把文件以新的 commit 形式推送到 DeviatedZHR/DeviatedZHR.github.io 仓库。

安全提示：为了安全，不要把 Token 硬编码在代码里。本页面只是在你本地浏览器里使用 你手动输入的 Token，不会把它保存到任何地方，也不会把它发到除 GitHub 以外的其他服务器。

1. GitHub 访问令牌

在 GitHub 个人设置中创建一个 Fine-grained personal access token，只勾选这个仓库的 contents: read & write 权限，然后复制粘贴到这里。

2. 目标路径与分支

默认会上传到 files/ 目录和 main 分支，你可以根据需要修改。

3. 选择文件并提交

挑选一个本地文件（例如 PDF 报告），然后填写一个简短的 commit 描述。
