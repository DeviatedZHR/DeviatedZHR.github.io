Curriculum Vitae & resources · 简历与资源

Central place for my CV, selected PDFs and other sharable files.
集中展示简历、精选 PDF 及其它可分享文件。

The following links are static files inside this GitHub repository, so you can update them simply by dropping new versions into the corresponding folders (for example cv/ or files/). This gives you a simple, Git-based “upload” workflow that works well with GitHub Pages.
下面的链接都是仓库中的静态文件，只要把新的 PDF 放进对应文件夹（如 cv/、files/）并提交即可完成“上传”。

Upload helper · 上传助手

Prefer a simple UI? You can also open a small helper page that uses the free GitHub REST API to push files into this repository (requires a personal access token with contents: write permission).
可使用上传助手页面，通过 GitHub REST API 将文件推送到本仓库（需具备 contents: write 权限的访问令牌）。

↗ Open GitHub API upload tool

Note: GitHub Pages is a static host, so uploads are managed via Git (commit + push) or via the GitHub web interface. This page is designed to make those files easy to find and nicely presented once they are in the repo.
